# Smelter Machine Core
<ItemImage id="meeptech:module_machine_core_smelter" />
The Smelter essentially acts as an upgraded furnace. It can perform any vanilla furnace recipe in addition to the furnace recipes custom to this mod. It requires a [Heating Core](../modules/heating_core.md).
## Blasting Upgrade
<ItemImage id="meeptech:module_upgrade_smelter_blasting" />
The blasting upgrade will allow the smelter to perform blasting recipes, which are twice as fast. It can still perform other recipes.
