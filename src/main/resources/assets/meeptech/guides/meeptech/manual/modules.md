# Modules
Modules are the fundamental building blocks of machines, defining the behavior of the machine. Modules go into module slots, which have a type that determines what modules can go there. Modules can themselves have submodule slots, meaning that modules are overall organized in a tree structure. A machine must have all required module slots filled to be completed. At the bottom layer of the tree are components, which are non-module items that will determine the stats of the machine.

Modules can also have upgrades, with the number of upgrade slots determined by the hull. Upgrade slots can go into any module.
