# Materials
Materials are a substance that can be used to make objects. In this mod, the materials you use are important, as using any material as a component for a module will impact the statistics of the machine.
# Material Statistics
## Thermal Conductivity/Resistance
Thermal conductivity or resistance represents how good or bad the material is at transferring heat.
## Melting Point
Melting point represents the maximum temperature it can withstand. Alternatively, a material can be flammable, making it unsuitable for high temperature applications.
## Tensile Strength
Tensile strength represent the force a material can withstand before breaking.
