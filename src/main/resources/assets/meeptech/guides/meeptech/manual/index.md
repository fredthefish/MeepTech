# Welcome
Welcome to **MeepTech**! This is a mod about industrialization using customizable machinery. This mod took major inspiration from GregTech and other technology mods, but it has significant deviations. See **[Getting Started](progression/getting_started.md)** for information on how to get started.
## Features
- **Machines** are modular and customizable. They are made out of **Modules** which determine their behavior.
- The **Materials** you use to build machines determine the statistics of the machine.
## This Book
This book can be accessed through holding a key (default keybind is G) over certain items, or by crafting it.
