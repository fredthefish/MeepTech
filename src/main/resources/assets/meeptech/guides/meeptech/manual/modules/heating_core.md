# Heating Cores
Heating cores provide heat to a machine.
# Heating Core Modules
<ItemImage id="meeptech:module_heating_core_solid_fuel" />
## Solid Fuel Module
The solid fuel module turns solid fuel, such as coal, into heat. It can use any regular fuel in addition to some items unique to it, like sugar cane, in order to achieve this. It requires a Firebox, which is just a [Box](../mechanics/forms.md#boxes). The firebox material determines the rate of the machine and the rate of fuel consumption.
